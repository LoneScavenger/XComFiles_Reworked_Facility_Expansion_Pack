# XcomFiles Reworked Facility Expansion Pack
   
    Submod for Solarius Scorch's X-COM Files mod
    By Scavenger

===== FAQ =====

 - What does this mod do?
This mod features garage facilities - small hangars for cars in early game, gives nearly all facilities 
bonus storage space and adds new hangars for mid-late game.
It changes the starting conditions a bit and slightly modifies the early-mid game research tree.

- Can I use these files for my own mod?
 This mod is not made by me. Since the original did not have any file usage restrictions or credit info, 
 if you want to use these files make sure to credit the original authors and me.

 - Why did you make this mod?
The submod was not updated for a long while and with X-Com Files updates it became outdated. 
It also had a couple of issues that affected gameplay significantly.

 - Is this submod on github?
https://github.com/LoneScavenger/XComFiles_Reworked_Facility_Expansion_Pack


 - Is this mod complete?
Most likely yes.

- Is this mod compatible with other submods?
It requires Brutal OXCE to allow different hangar types.
Other mods that introduce new crafts may not work properly.
My Scavenger's Additions mod has been made compatible.
